# Time Zone Comparison Application

A web application that displays a time comparison table between multiple cities and time zones.

## Features

- **Time Zone Comparison**: View UTC time alongside local times for 9 different locations:
  - Manila (UTC+8)
  - Gothenburg (UTC+1/+2)
  - Chennai (UTC+5:30)
  - Hanoi (UTC+7)
  - Dubai (UTC+4)
  - Saudi Arabia (UTC+3)
  - London (UTC+0/+1)
  - Japan (UTC+9)
  - South Korea (UTC+9)

- **Visual Features**:
  - Color-coded rows by day of the week
  - Highlighted current time row
  - Visual indicators for day changes, business hours, and evening times
  - Responsive design for different screen sizes

- **Navigation**:
  - Navigate between days with Previous/Next buttons
  - Quick "Today" button to jump to current date
  - Auto-refresh to keep current time highlighted

## Usage

Simply open `index.html` in a web browser. No installation or build process required.

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and visual design
- `script.js` - Time calculation and table generation logic

## Browser Compatibility

Works in all modern browsers that support ES6 JavaScript and CSS Grid/Flexbox.
